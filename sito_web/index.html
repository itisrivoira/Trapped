STORIA

Ex studente del Denina legge un articolo di giornale nel quale si ricorda il decimo anniversario dalla chiusura dell’istituto a causa dell’inagibilità. Durante questo periodo la scuola è stata lasciata in degrado, così è diventata un posto malsano dove la notte vanno a dormire dei senzatetto. Nell'ultimo periodo questi ultimi hanno dovuto lasciare lo stabile perché hanno riscontrato rumori e comportamenti strani che fanno presagire la presenza di un fantasma.

Il giovane curioso, attratto da questa stravagante vicenda, decide una notte di introdursi nell’edificio per indagare; quella sera, non appena superata la porta d’ingresso questa gli si chiude dietro e non riuscirà più ad aprirla, rimanendo quindi bloccato all’interno.

Dopodiché appare un fantasma, portandolo a svenire dalla paura; si risveglia poi nella sua vecchia classe chiusa e per uscire dovrà trovare la chiave rispondendo ad una serie di enigmi.

Uscendo troverà un corridoio con diverse stanze ai lati ed diversi piccoli fantasmi di disturbo i quali non dovrà incrociare per non subire dei de potenziamenti. Inoltre il ragazzo potrà trovare anche degli oggetti speciali per ottenere degli aiuti negli enigmi, e delle parti di giornale che lo aiuteranno.

Il ragazzo troverà poi altre chiavi per sbloccare le successive stanza che potrà prendere rispondendo a ad altri enigmi; nel momento in cui troverà risponderà all’ultimo enigma nell’ultima stanza e otterrà l’oggetto speciale rinchiuso in un cassetto, verrà rapito dal nemico e... Finisci il gioco per scoprirlo.

DOWNLOAD

COMANDI NECESSARI

WEB APP:


sudo apt-get update

Servirà per aggiornare la lista dei pacchetti contenuti nel repository locale, ovvero la lista dei “programmi” che sei in grado di scaricare


sudo apt install nodejs

Servirà per lanciare l’installazione vera e propria di nodejs


Controlla se la versione dei pacchetti è corretta con:

node --version

Versione Node JS >= 14.0.0

npm --version

Versione di npm >= 5.6


COMANDI PER UTILIZZARE LA WEB APP:

Per poter usare la web app devi prima istallare alcuni moduli aggiuntivi non di default in npm


Quindi apri la cartella con il download della web app e perciò vai in:

./Web App


Apri il terminale e digiti questi seguenti comandi:

npm install react-router-dom


npm install react-typewriter


Per avviare la web app devi aprire la cartella:

./Web App


Apri il terminale e digita:

npm start


COMANDI PER INSTALLARE PYTHON:


python --version

Verifica la presenza o meno di python sul proprio pc


sudo apt-get install python3.8

Comando per installare python


COMANDI PER INSTALLARE LA LIBRERIA PYGAME:


sudo apt-get install python-pygame


COMANDI PER AVVIARE IL GIOCO:


Apri la cartella con tutto i file python:

./Python


Aprire il terminale e digitare:

python3 main.py
